Italian Non-Dom Tax Regime: Spatial Equilibrium Analysis

Fiscal Amenities and Spatial Inequality

Research Question: How did Italy’s 2017 non-dom tax regime affect regionalhousing prices and housing-burden inequality?

Methodology: Continuous Difference-in-Differences with:

  • Regional Amenity Index (tourism, crime, coastal access);

  • Event study diagnostics;

  • Pre-trend validation.

3. Amenity Index construction

The Amenity Index combines pre-2017 regional characteristics: 1. Foreign tourist nights per capita (tourism intensity) 2. Crime rate per 100,000 inhabitants (inverted) 3. Coastal dummy (binary indicator) Formula: AmenityIndex = Z_Tourism - Z_Crime + Coastal

# 3. Time-invariant amenity index (z-scores from 2015-2019 averages)
final <- final %>%
  mutate(
    z_tourism    = as.numeric(scale(foreign_pc_avg)),
    z_crime      = as.numeric(scale(crime_avg)),
    AmenityIndex = z_tourism - z_crime + coastal_dummy,
    Post = ifelse(Year >= 2017, 1, 0)
  )

6. EVENT STUDY DIAGNOSTICS

# The continuous event study validates the parallel trends assumption and
# provides year-by-year treatment effects.

analysis_data <- analysis_data %>%
  mutate(rel_year = Year - 2017)

# Event study: Housing Prices
event_hpi <- feols(
  log_HPI_average ~ 
    i(rel_year, AmenityIndex, ref = -1) | territory_code + Year,
  data = analysis_data,
  cluster = ~territory_code
)

# Extract event study coefficients for plotting
event_coefs <- broom::tidy(event_hpi) %>%
  filter(grepl("AmenityIndex", term)) %>%
  mutate(
    rel_year = as.numeric(gsub(".*rel_year::(\\-?\\d+).*", "\\1", term)),
    estimate = estimate,
    std.error = std.error,
    ci_lower = estimate - 1.96 * std.error,
    ci_upper = estimate + 1.96 * std.error
  ) %>%
  filter(!is.na(rel_year))

fig6_event_study <- ggplot(event_coefs, aes(x = rel_year, y = estimate)) +
  geom_point(size = 3, color = "darkblue") +
  geom_errorbar(aes(ymin = ci_lower, ymax = ci_upper), width = 0.2, color = "darkblue") +
  geom_hline(yintercept = 0, linetype = "dashed", color = "red") +
  geom_vline(xintercept = -0.5, linetype = "dashed", color = "black", alpha = 0.5) +
  theme_minimal() +
  theme(panel.grid.major.x = element_line(color = "grey90")) +
  labs(
    title = "Continuous Event Study: Effect of Amenity on Housing Prices",
    subtitle = "Pre-2017 coefficients should be near zero (parallel trends); dashed line at 2017 reform",
    x = "Years Relative to 2017 Reform",
    y = "Coefficient (95% CI)",
    caption = "Negative rel_year = before reform; Positive = after reform"
  )
